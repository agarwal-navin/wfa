Install & use offline

‹ Back

📱 iPhone / iPad (Safari)

  1. Open this site in Safari (Add to Home Screen isn't available in other browsers on iOS).
  2. Tap the Share button (the square with an up arrow).
  3. Scroll down and tap Add to Home Screen.
  4. Tap Add. The Wilderness First Aid icon appears on your home screen.
  5. Open it from that icon - it runs full-screen like an app.

🤖 Android (Chrome)

  1. Open this site in Chrome.
  2. Tap the ⋮ menu (top-right).
  3. Tap Install app (or Add to Home screen).
  4. Confirm with Install. The icon is added to your home screen / app drawer.

📶 Make sure it works offline

The app saves itself to your device the first time it loads, so it can run with no signal (great for the backcountry). To be sure it's cached:

  1. Install it using the steps above.
  2. Open the app while you still have internet.
  3. Tap Open the guide and browse a few screens (Assessment, Injuries, Glossary) so everything downloads.
  4. Give it a few seconds, then fully close the app.
  5. Test it: turn on Airplane Mode and reopen - it should load normally.
Tip: After an update, open the app once with internet and reload so the newest version is cached. If something looks stale, close and reopen it.
